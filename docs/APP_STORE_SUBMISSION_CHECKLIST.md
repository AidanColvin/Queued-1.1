# Queued App Store Submission Checklist

## Blocking items
- [ ] Confirm MovieLens and all dataset licenses allow your intended App Store distribution model.
- [ ] Publish a public privacy policy URL.
- [ ] Complete App Privacy answers in App Store Connect.
- [ ] Verify account deletion works in production.
- [ ] Verify password reset and email verification work in production.
- [ ] Add Sign in with Apple if the app offers qualifying third-party sign-in methods.
- [ ] Prepare screenshots, subtitle, description, keywords, support URL, and marketing URL.
- [ ] Test an archive on a physical iPhone.
- [ ] Upload a build to TestFlight.
- [ ] Fix all crashes from TestFlight before submission.

## Build checks
- [ ] Bundle identifier set.
- [ ] Version and build number set.
- [ ] App icon and launch screens complete.
- [ ] Production API URL configured.
- [ ] Privacy strings reviewed in Info.plist if needed.
- [ ] App category selected.
- [ ] Age rating answered.

## Review notes
- [ ] Explain demo/sample data clearly if applicable.
- [ ] Explain login requirements, if any.
- [ ] Explain imported data sources and third-party attribution.
